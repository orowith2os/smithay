pub unsafe fn make_current_with_surface(
    &self,
    surface: &EGLSurface
) -> Result<(), MakeCurrentError>

Makes the OpenGL context the current context in the current thread with a surface to read/write to.

Safety

This function is marked unsafe, because the context cannot be made current on another thread without being unbound again (see EGLContext::unbind).
